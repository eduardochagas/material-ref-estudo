FORMATAÇÂO DENTRO DAS strings

\ - usado dentro da string para "quebrar a string" para continuar escrevendo na linha seguinte

%d - especificação para números decimais.

%s - especificação para strings.

%e - especificação para exponenciação (n ** n).

%x - especificação para números hexadecimais (base 16).

%f - especificação para números de ponto flutuante.

%20s\n - preenche uma string até 20 caracteres e exibe um caractere de nova linha.

%07d - colocar o zero depois do sinal de porcentagem significa: preecha com zero
 os caracteres que sobraram após inserido o valor. 

%( ) - especifica os valores para a(s) formataçôes dentro da(s) string(s) e
a ordem dos valores inseridos aqui será inserido na mesma ordem dentro
da(s) formatações na(s) string(s).
